> making a quick to set-up video streaming service to
> provide your local videos on your network in a better way.
...

## Installation

Add this line to your application's Gemfile:

    gem 'biscope'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install biscope

## Usage

To start Biscope Server
$ biscope play
